    Build It¶

    First, a few things need to be done.

    Create a gh-pages branch

    Install wkhtmltopdf.

    I’ll assume there is a Python install with virtualenv. Get one setup for this and pull in the dependencies:

    $ make venv
$ source env/bin/activate

    Now create a pdf:

    (env)$ make pdf

    Or push the latest changes to github:

    (env)$ make publish

    Or do whatever else sphinx can do.
